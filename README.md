# Book Tracker

**Автор:** Алмаз Анварович

## Описание
Приложение для учёта прочитанных книг с графическим интерфейсом на Python (tkinter).  
Позволяет добавлять книги, фильтровать по жанру и количеству страниц, сохранять и загружать данные в JSON.

## Требования
- Python 3.8 или выше
- Библиотека tkinter (встроена в Python)

## Запуск
1. Откройте терминал в папке с программой
2. Выполните команду:
python main.py

text

## Как пользоваться
1. **Добавление книги:** заполните поля (Название, Автор, Жанр, Страниц) и нажмите "Добавить книгу"
2. **Фильтрация:** 
- Введите жанр для фильтрации по жанру
- Введите число в поле "Страниц >" чтобы показать книги длиннее этого числа
- Нажмите "Сбросить фильтр" чтобы показать все книги
3. **Сохранение:** нажмите "Сохранить в JSON" — данные сохранятся в файл `books.json`
4. **Загрузка:** нажмите "Загрузить из JSON" — загрузит ранее сохранённые книги

## Пример использования
1. Добавьте книгу: "1984", "Джордж Оруэлл", "Роман", 328
2. Добавьте книгу: "Война и мир", "Лев Толстой", "Роман", 1225
3. Добавьте книгу: "Краткие ответы на большие вопросы", "Стивен Хокинг", "Научпоп", 256
4. Примените фильтр по жанру "Роман" → увидите только первые две книги
5. Примените фильтр "Страниц > 500" → увидите только "Войну и мир"

## Проверка корректности ввода
- Все поля обязательны для заполнения
- В поле "Страниц" можно вводить только числа
- При попытке добавить пустую книгу или буквы в страницах появится ошибка

## Файлы программы
- `main.py` — основной код приложения
- `books.json` — файл с сохранёнными книгами (создаётся автоматически при сохранении)
- `README.md` — этот файл с инструкцией
